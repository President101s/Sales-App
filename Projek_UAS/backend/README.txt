How to run? 
Simply just go to the 'dist' folder, and there you will find a single file named main.exe, and run it. 
Actually the file with this format can be run as simple as double clicking on it, but because the app is flask (which running on console), so you have to type the 'main.exe' on the console as well.

How to use the sales db?
You need to create a db first in the server with same name, which is "sales".
